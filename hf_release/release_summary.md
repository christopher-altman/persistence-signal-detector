# UCIP Phase I Release Summary

This Hugging Face bundle is the bounded reproducibility companion for the retained UCIP Phase I artifact layer aligned to the current submission-facing state associated with `arXiv:2603.11382`. It packages the normalized retained results, the artifact-authority layer, the paper-aligned configs and figure exports, and the minimal notebooks/scripts needed to inspect how the reported claims map onto the retained evidence surface.

## Included

- Normalized retained JSON artifacts from `results/`
- Artifact authority files covering scope, provenance, canonicality, and overlap handling
- Frozen and paper-aligned configs
- Extracted frozen Phase I thresholds
- Canonical paper figure PDFs and retained tables
- Reproducibility notebooks and scripts
- Release-level provenance manifests

## Exclusions and non-claims

- No standalone raw trajectory corpus
- No standalone label files
- No standalone split files
- No claim of deployment readiness
- No claim of consciousness or sentience detection
- No claim that every paper figure is exactly regenerable from public JSON alone
